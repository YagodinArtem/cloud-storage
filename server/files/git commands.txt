__________________________________ Stash

Git Stash - специальное пространство, которое хранится только на локальном репо
и записывает все текущие изменения, отменяя состояние репозитория.

git stash (все изменения выгружены во временное хранилище)

git stash list

git stash push new-feature.go (добавляет новый файл к уже существующему стешу)

git stash pop (stash name) (развернуть стеш на текущей ветке)

__________________________________ Squash

git merge squash-dev --squash (для того, чтобы записать все изменения в рамках 1 коммита)

git rebase -i HEAD~3 (количество коммитов которые мы хотим объединить)

Открывает режим слияния коммитов с различными функциями
